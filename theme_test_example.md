# Theme System 2D Grid Examples

## Office Theme Grid Example

```
#########
#.......#
#.......D
#.......#
#   O   #
#.......#
#D......#
#.......#
#########
```

Legend:

- `#` = Drywall (walls)
- `.` = Carpet Floor
- `D` = Office Door  
- `O` = Office Window
- ` ` = Tile Floor (corridors)

## Dungeon Theme Grid Example  

```
████████████

█·········█
█·········☩
█·▓▓▓▓▓▓▓▓█
```

Legend:

- `█` = Stone Wall
- `·` = Stone Floor
- `☩` = Iron Gate (door)
- `▓` = Cobblestone (corridor)
- `═` = Stone Stairs

## Castle Theme Grid Example

```
▌▌▌▌▌▌▌▌▌▌▌▌▌
▌⬜⬜⬜⬜⬜⬜⬜⬜⬜⬜▌

▌⬜▌▥▥▥▥▥▥▌⬜⬜▌
▌⬜▌▥⬜⬜⬜▥▌⬜⬜▌
▌⬜▌▥⬜⬌⬜▥▌⚿⬜▌
▌⬜▌▥⬜⬜⬜▥▌⬜⬜▌
▌⬜▌▥▥▥▥▥▥▌⬜⬜▌

```

Legend:

- `▌` = Castle Wall (massive stone)
- `⬜` = Stone Tiles (floor)
- `⚿` = Great Door
- `⬦` = Arrow Slit (window)
- `▥` = Flagstone (corridor)
- `⬌` = Stone Steps (stairs)

These ASCII representations provide immediate visual feedback for level designers while editing in 2D grid mode, with seamless conversion to 3D meshes for final visualization.
